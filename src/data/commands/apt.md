```bash
# パッケージのリストを更新する
sudo apt update
# インストール済みのパッケージを更新する
sudo apt upgrade
# キャッシュを削除する
sudo apt clean
```

## apt-mark

```bash
# 手動でインストールしたパッケージを列挙する
apt-mark showmanual
```
